# 🍔 Azarkhsh Fast Food Landing Page

A vibrant and engaging landing page for **Azarkhsh Fast Food**, built with **HTML, CSS, and JavaScript**. This project showcases a modern design and interactive elements to present the restaurant's offerings effectively.

As an early-stage project, it demonstrates foundational web development skills with a focus on user experience and visual appeal.

## ✨ Key Features

*   **Hero Section:** A striking introductory section with a captivating image and clear call-to-action.
*   **Menu Showcase:** Deliciously presented menu items with descriptions and prices.
*   **Interactive Animations:** Engaging scroll-based animations that bring the page to life as users explore.
*   **Responsive Design:** Optimized for seamless viewing across all devices (desktops, tablets, and mobile phones).
*   **Clean HTML, CSS, and JS:** Well-structured and maintainable code.
*   **Contact Information:** Easy access to restaurant location and contact details.

## 🚀 Technologies Used

*   HTML5
*   CSS3
*   JavaScript (Vanilla JS)

## ⚙️ How to View

Simply open the `index.html` file in your web browser to explore the landing page.
(No server or special installation required)

## 🎯 Project Goal

To create a visually appealing and interactive landing page that effectively introduces the Azarkhsh Fast Food brand and encourages customer engagement.
